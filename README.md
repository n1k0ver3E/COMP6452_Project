# Foodie Chain

## Client
**Requirements**
- NPM
- Node v16

**Run App Locally**

1. Navigate to the client folder

    ```
    $ cd client
    ```

2. Run app (Port 3000)

    ```
    $ npm run start
    ```


## Server
**Requirements**
- NPM
- Node v16

**Run App Locally**

1. Navigate to the server folder

    ```
    $ cd server
    ```

2. Run app (Port 5000)

    ```
    $ npm run server
    ```
